Please visit game's web page at http://toppler.wz.cz where you can download
the latest versions of the program and source codes and read the information
about the game.

What's new in version 1.1
-------------------------
- fixed high CPU usage
- several minor changes (which influence mainly source codes)

Minimum system requirements
---------------------------
- PC with Windows 98/ME/XP
- DirectX 7 or higher
- optionally sound card


Uninstall info
--------------
Simply delete all installed files, the program doesn't record anything to registry.
